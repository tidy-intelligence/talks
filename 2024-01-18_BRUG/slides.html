What is Quarto?

An open-source, scientific & technical publishing system


Quarto Schematic, Artwork by Allison Horst.

Weave together narrative & code

Produce elegantly formatted output as documents, web pages, blog posts, books and more



Using a variety of authoring tools!

Quarto Authoring Tools.

Just like Rmarkdown …

… but much more powerful!


  • Consistent implementation of attractive & handy features across outputs: tabsets, code-folding, syntax highlighting, etc.
  • Features & defaults that enable good practices for producing accessible documents
  • Natively multi-lingual, supporting for multiple languages like Python, Julia, Observable

Move between formats easily

Document

document.qmd

title: "My Document"
format: pdf

Presentation

slides.qmd

title: "My presentation"
format: revealjs

Dashboard

dashboard.qmd

project:
  type: dashboard

Website

_quarto.yml

project:
  type: website

website: 
  navbar: 
    left:
      - page-one.qmd

Book

_quarto.yml

project:
  type: book

book:
  title: "My Book"
  author: "Jane Doe"
  date: "18/01/2024"
  chapters:
    - index.qmd

Include code & plot with R

library(ggplot2)
library(palmerpenguins)

penguins <- palmerpenguins::penguins

ggplot(penguins,
       aes(x = bill_length_mm, y = bill_depth_mm, color = species, shape = species)) +
  geom_point() +
  geom_smooth(method = "lm") +
  labs(x = "Bill length (mm)", y = "Bill Width (mm)",
       title = "Bill Length vs. Bill Width", color = "Species", shape = "Species")

Replicate the same with Python

from plotnine import *
from palmerpenguins import load_penguins

penguins = load_penguins()

(ggplot(penguins, 
        aes(x = "bill_length_mm", y = "bill_depth_mm", color = "species", shape = "species"))
  + geom_point()
  + geom_smooth(method = "lm")
  + labs(x = "Bill Length (mm)", y = "Bill Width (mm)", 
         title = "Bill Length vs. Bill Width", color = "Species", shape = "Species"))
